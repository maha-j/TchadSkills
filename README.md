# 🇹🇩 TchadSkills - Plateforme de Développement des Compétences

TchadSkills est une plateforme e-learning moderne conçue spécifiquement pour le contexte tchadien, permettant la formation continue dans des domaines clés tels que le développement web, le marketing digital et l'entrepreneuriat.

## 🚀 Fonctionnalités Clés
- **Backend Robuste** : Propulsé par Django & Django REST Framework.
- **API REST** : Prête pour l'intégration mobile ou front-end moderne.
- **Système d'Apprentissage** : Gestion des cours, leçons et catégories.
- **Paiements & Inscriptions** : Structure prête pour l'intégration de Mobile Money.
- **Administration** : Interface complète pour la gestion du contenu.

## 🛠️ Installation Rapide

Pour lancer le projet sur votre machine locale :

1. **Cloner le projet**
   ```bash
   git clone https://github.com/maha-j/TchadSkills.git
   cd TchadSkills
   ```

2. **Installer et configurer**
   Nous avons inclus un script d'automatisation pour vous faciliter la tâche :
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

3. **Lancer le serveur**
   ```bash
   python manage.py runserver
   ```

## 📂 Structure du Projet
- `core/` : Logique métier, modèles et API.
- `tchadskills_project/` : Configuration globale de l'application.
- `templates/` : Interface utilisateur (HTML).
- `static/` : Ressources statiques (JavaScript, CSS).
- `seed_data.py` : Script pour peupler la base avec des données réelles.

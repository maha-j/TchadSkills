# Setup Guide

## Prerequisites
Before you begin, ensure you have the following installed:
- **Node.js** (version X.X or higher)
- **npm** (version X.X or higher)
- **Git**

## Cloning the Repository
To clone the repository, run the following command:

```bash
git clone https://github.com/maha-j/TchadSkills.git
cd TchadSkills
```

## Installing Dependencies
Navigate to the project directory and install the required dependencies:

```bash
npm install
```

## Setting Up the Environment Variables
Create a `.env` file in the root of the project and add the necessary configurations:

```.env
# Example environment variables
DATABASE_URL=your_database_url
API_KEY=your_api_key
```

## Running the Application
To start the application, use:

```bash
npm start
```

## Running Tests
To run the test suite, execute:

```bash
npm test
```

## Troubleshooting
- If you encounter issues, consult the logs and ensure your environment variables are correctly set up.
- Check for any dependency issues by running `npm install` again.

## Conclusion
You should now have the application up and running. For further assistance, please consult the documentation or open an issue on the GitHub repository.
